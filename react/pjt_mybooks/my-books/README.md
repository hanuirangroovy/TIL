- react-router-dom v6 업그레이드 되면서 달라진 것들
  - 참고사이트
    - https://velog.io/@kcdoggo/Switch-is-not-exported-from-react-router-dom-%EC%97%90%EB%9F%AC
    - https://velog.io/@soryeongk/ReactRouterDomV6
- `npm i react-error-boundary`
